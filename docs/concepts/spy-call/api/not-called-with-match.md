# `spyCall.notCalledWithMatch(arg1, arg2, ...);`

Returns `true`, when the spyt call did not receive matching arguments.
This behaves the same as [`spyCall.notCalledWith(sinon.match(arg1), sinon.match(arg2), ...)`](./not-called-with).
