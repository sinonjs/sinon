# `sinon.match.map`

Requires the value to be a `Map`.

## `sinon.match.map.deepEquals(map)`

Requires a `Map` to be deep equal another one.

## `sinon.match.map.contains(map)`

Requires a `Map` to contain each one of the items the given map has.
