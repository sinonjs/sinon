# `sinon.match.set`

Requires the value to be a `Set`.

## `sinon.match.set.deepEquals(set)`

Requires a `Set` to be deep equal another one.

## `sinon.match.set.contains(set)`

Requires a `Set` to contain each one of the items the given set has.
